gamlss.random(gamlss)R Documentation

Support for Function random()

Description

This is support for the function random(). It is not intended to be called directly by users. The function gamlss.radom is similar to the GAMLSS function gamlss.ra.

Usage

gamlss.random(x, y, w, df = NULL, lambda = 0)

Arguments

x the explanatory design matrix
y the response variable
w iterative weights
df effective degrees of freedom
lambda the smoothing parameter

Value

Returns a list with

y the fitted values
residuals the residuals
var the variance of the fitted values
lambda the final lambda, the smoothing parameter
coefSmo with value NULL

Author(s)

Mikis Stasinopoulos, based on Trevor Hastie function gam.random
