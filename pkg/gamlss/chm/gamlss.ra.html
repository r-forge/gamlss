gamlss.ra(gamlss)R Documentation

Support for Function ra()

Description

This is support for the random effect smoother function ra() . It is not intended to be called directly by users. The function gamlss.ra is similar to the GAMLSS function gamlss.random. Both functions can be used with the same effect.

Usage

gamlss.ra(x, y, w, df = NULL)

Arguments

x the explanatory design matrix
y the response variable
w iterative weights
df effective degrees of freedom

Details

Value

Returns an list with

fitted.values fitted values
residuals residuals
var variances of the fitted values
nl.df the trace of the smoothing matrix
lambda the value of the smoothing parameter
coefSmo the coefficients from the smoothing fit
varcoeff the variance of the coefficients

Note

This is an experimental function and should be used with care
