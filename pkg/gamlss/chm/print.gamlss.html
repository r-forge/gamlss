print.gamlss(gamlss)R Documentation

Prints a GAMLSS fitted model

Description

print.gamlss is the GAMLSS specific method for the generic function print which prints objects returned by modelling functions.

Usage

## S3 method for class 'gamlss':
print(x, digits = max(3, getOption("digits") - 3), ...)

Arguments

x a GAMLSS fitted model
digits the number of significant digits to use when printing
... for extra arguments

Details

Value

Prints a gamlss object

Author(s)

Mikis Stasinopoulos d.stasinopoulos@londonmet.ac.uk, Bob Rigby r.rigby@londonmet.ac.uk and Calliope Akantziliotou

Examples

data(aids)
h<-gamlss(y~poly(x,3)+qrt, family=PO, data=aids) 
print(h) # or just h
rm(h)
